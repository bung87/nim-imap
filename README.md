RFC3501 client implementation in nim
====================================

## Instalation
### Nim package manager

```bash
nimble install https://github.com/forlan-ua/nim-imap.git
```

### Install from source code

```bash
git clone https://github.com/forlan-ua/nim-imap.git
cd nim-imap
nimble develop -y
```

## Run tests

```bash
git clone https://github.com/forlan-ua/nim-imap.git
cd nim-imap
nimble install nake -y
nake
```

## Generate docs

```bash
git clone https://github.com/forlan-ua/nim-imap.git
cd nim-imap
nimble install nake -y
nake doc
```
